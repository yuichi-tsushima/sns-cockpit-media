# sns-cockpit-media

yutaka_ehon（ゆたか絵本ラボ）・japantaleteller（Kon昔話）のSNS投稿用画像を
公開URL化するための静的ホスティング専用リポジトリ。

- `images/<brand>/<filename>` に画像を配置
- GitHub Pages経由で `https://yuichi-tsushima.github.io/sns-cockpit-media/images/<brand>/<filename>` として公開
- sns-cockpit（apps/sns-cockpit/）の投稿本文とは別リポジトリ。ここは画像配信専用。
